## 2. 비전 및 목표

경주 문화선도산단은 신라 천년 고도(古都)의 문화 자산과 첨단 산업을 융합한 **대한민국 최초의 문화선도형 산업단지**입니다.

단순 제조 산단이 아닌 **문화·관광·스마트시티·AI·안전**이 통합된 복합 거점으로,
연간 방문객 **10만 명** 유치 및 입주기업 **100개사** 달성을 중기 목표로 합니다.

### 4대 추진 방향

| # | 방향 | 설명 |
|---|------|------|
| 01 | **문화 + 산업의 융합** | 신라 천년 문화 자산과 첨단 산업이 공존하는 복합 단지 조성 |
| 02 | **스마트시티 기반 구축** | 스마트주차·스마트안전·데이터 기반 운영 인프라 선진화 |
| 03 | **AI 기반 기업지원** | AI 챗봇·정책데이터 분석·맞춤형 기업지원 서비스 제공 |
| 04 | **산업관광 허브화** | 경주 산업관광 거점으로서 연간 방문객 10만명 목표 |

## 3. 플랫폼 서비스 구성

CMV Platform은 산단 운영에 필요한 6개 핵심 서비스를 단일 포털로 통합 제공합니다.

### 3-1. 행사·프로그램

- 문화·산업·네트워킹 행사 통합 조회 및 온라인 신청
- 행사 상태 실시간 관리 (모집중 / 진행중 / 완료)
- 참가자 관리, 정원 현황 표시
- 무료·유료 행사 구분 안내

### 3-2. 산업관광 / 현장 견학

- 스마트팩토리, 문화복합공간, EV 모빌리티 현장 견학 프로그램
- 대상별 프로그램 구분: 학교단체 / 기업연수 / 일반관광
- 온라인 예약 신청, 희망 일정·인원 접수
- 프로그램 유형: 공장견학 / 체험형 / 산업문화 / 단체견학

### 3-3. 공간·시설 예약

- 회의실 / 교육장 / 전시홀 / 행사장 / 홍보관 온라인 예약
- 수용 인원, 면적, 운영 시간, 사용 요금 조회
- 예약 신청 → 관리자 승인 → 확정 워크플로우

### 3-4. 기업지원 / 공지사항

| 카테고리 | 내용 |
|---------|------|
| 일반공지 | 산단 운영 공지 |
| 지원사업 | 중소기업·스타트업 대상 보조금·지원사업 안내 |
| 입주기업 안내 | 입주사 전용 안내 및 운영 정보 |
| 행사안내 | 행사·프로그램 개최 공지 |
| 행정/민원 | 민원 처리, 행정 안내 |
| 공공사업 | 국가·지자체 연계 공공사업 안내 |

### 3-5. FAQ / 민원 안내

- 자주 묻는 질문 카테고리별 제공
- 키워드 검색 기능
- AI 챗봇 연계 즉시 답변 서비스

### 3-6. AI 안내 어시스턴트

- 행사·견학·공간예약·기업지원·FAQ 전반 24시간 대응
- 자연어 질의 → 정확한 정보 안내
- 바이칼시스템즈 AI 엔진 탑재
- 평균 응답 1초 이내

## 4. 스마트 인프라 연계

### 4-1. 스마트주차 시스템 (스마트한)

- 실시간 주차 구역별 현황 모니터링 (사용 중 / 여유 / 전체)
- 행사별 주차 수요 예측 및 관제
- 주차장 코드별 가용 현황 대시보드 제공
- 영상관제 시스템 통합 연계

### 4-2. 스마트안전 시스템 (스마트아이넷)

- BLE / UWB 기반 스마트태그 발급 및 관리
- 방문객·근로자·행사 참여자 위치 기반 안전 모니터링
- 긴급호출(SOS) / 구역 경보 / 태그 분실 감지 실시간 처리
- 미처리 경고·긴급 이벤트 즉시 알림 및 대시보드 표시
